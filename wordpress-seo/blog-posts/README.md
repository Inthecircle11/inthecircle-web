# Blog Posts for inthecircle.co – SEO & Keyword Ranking

These posts target the main keywords so **inthecircle.co** can rank for creator- and app-related searches. Add them as **WordPress Posts** (not pages) so they appear in your blog feed and sitemap.

---

## Did we index this? Do I need to do anything?

**Nothing is auto-indexed from this repo.** The blog content lives here as files; search engines only see what’s **published on your live site**. You need to do the following from your end.

### 1. Publish the 9 blog posts on WordPress (inthecircle.co)

**Fastest:** Use the **one-file import** so all 9 posts are created at once.

- In WordPress: **Tools → Import → WordPress** → upload **`inthecircle-blog-import.xml`** (in the `wordpress-seo` folder; generated by `node scripts/generate-wxr.js`). Assign author and run the importer.
- Then set each post’s **SEO meta title**, **meta description**, and **focus keyword** in Yoast or Rank Math (see frontmatter in each `.md` or the table below).

Until the posts exist on the live site (import or manual), they **cannot be indexed**. Do this first.

### 2. Make sure your WordPress sitemap includes posts

- If you use **Yoast SEO** or **Rank Math** or **All in One SEO**: enable the XML sitemap and ensure **Posts** are included (not only pages).
- Your sitemap URL is usually: `https://inthecircle.co/sitemap_index.xml` (Yoast) or `https://inthecircle.co/sitemap.xml` (Rank Math).

### 3. Submit your sitemap in Google Search Console

- Go to [Google Search Console](https://search.google.com/search-console).
- Select the property for **inthecircle.co** (add the property if you haven’t).
- Open **Sitemaps** and submit: `sitemap_index.xml` or `sitemap.xml` (whatever your SEO plugin uses).
- After you publish the new posts, you can use **URL Inspection → Request indexing** for a few key post URLs to speed up discovery.

### 4. App site (app.inthecircle.co)

- The Next.js app already has `/sitemap.xml` and `/robots.txt` in the repo and on the live deploy.
- If you haven’t already, add **https://app.inthecircle.co** as a property in Google Search Console and submit its sitemap: `https://app.inthecircle.co/sitemap.xml`.

### 5. Optional: Bing Webmaster Tools

- Add inthecircle.co (and optionally app.inthecircle.co) and submit the same sitemap URL for Bing indexing.

**Summary:** The code and content in the repo are set up for SEO, but **you** must publish the posts on WordPress, confirm the sitemap includes them, and submit that sitemap (and the app sitemap) in Google Search Console so everything gets indexed.

## How to Add These to WordPress

### For each post (e.g. `01-best-creator-networking-app-2026.md`):

1. **Create a new Post**  
   WordPress → Posts → Add New.

2. **Set the URL slug**  
   In the right sidebar (or under the title), set the permalink/slug to the value in the file’s frontmatter (the `---` block at the top).  
   Example: `best-creator-networking-app-2026`.

3. **Set SEO title and description (Yoast / Rank Math / All in One SEO)**  
   Use the “Meta title” and “Meta description” from the frontmatter.  
   Example:  
   - **Meta title:** Best Creator Networking App 2026 – Connect & Collaborate | inthecircle  
   - **Meta description:** The best creator networking app in 2026 helps founders, YouTubers & streamers connect. No ads, real collaborations. See why inthecircle leads.

4. **Paste the post body**  
   Copy everything **below** the frontmatter (below the second `---`). Paste into the WordPress editor.  
   - If you use the **block editor**: paste as normal; headings and links will become blocks.  
   - If you use **Classic Editor**: the markdown `#` and `##` will show as plain text—replace `# ` with one H1, `## ` with H2, `### ` with H3, or switch to block editor for that post.

5. **Set focus keyword**  
   In your SEO plugin, set the “focus keyword” to the one in the frontmatter (e.g. `creator networking app`).

6. **Publish**  
   Publish or schedule. Repeat for all 9 posts.
