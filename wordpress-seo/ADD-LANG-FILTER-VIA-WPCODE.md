# Add blog language filter via WPCode (no plugin upload)

Your site already has **WPCode** (it appears in the admin bar). You can enable the blog language filter in about 1 minute:

1. In WordPress admin go to **WPCode** → **Code Snippets** → **Add Snippet**.
2. Choose **Add Your Custom Code (New Snippet)**.
3. Snippet type: **PHP Snippet**.
4. Copy the **entire contents** of `snippet-blog-lang-filter.php` (in this folder) and paste into the code box.
5. Set **Location** to **Run Everywhere**.
6. Click **Save Snippet** then **Activate**.

After that, open https://inthecircle.co/blog/ and you should see the filter bar (All | عربي | English) at the top of the posts.

No plugin file upload or FTP needed.
